# design/reference — 래디얼 메뉴 원본 참고 자료

## radialmenu.html (권위 원본 — 이 파일이 디자인의 진실)

사용자가 직접 제공한 참고 구현(타 디자인 AI 제작, 2026-07). **래디얼 메뉴의 외형·모션은 이
파일을 기준으로 이식한다.** 핵심 포인트:

- **오브 = 살아있는 유리 칩**: 완전한 원 금지. 80×56(px≈dp) 라운드 사각, border-radius 가
  20→28→20→16→20 으로 **끊임없이 morph**(오브별 랜덤 주기 3~5s). 채움 rgba(20,60,180,0.12),
  림 1.5px rgba(180,240,255,0.55)(은은하게 — 강조 테두리 금지), 안쪽 링 1px @0.20 inset 4px,
  좌상단 광택 타원, 외곽 글로우 2겹(20px rgba(120,200,255,.70) / 50px rgba(80,150,255,.40)).
- **라벨은 칩 아래**: 12px #cdeeff bold + 시안 글로우 3겹.
- **선은 곡선**: 배지→오브 스포크 5 + 인접 오브 호 4. 중점을 배지 쪽으로 24px 당긴
  2차 베지어. 양 끝은 배지/오브 가장자리까지 트리밍. glow #4da8ff@.30(4px, blur) +
  crisp #cdeeff@.55(1.2px). 오브 터치 시 연결선 하이라이트(0.95/1.8px).
- **빛 점(travel dot)**: 각 곡선을 따라 흐름. r 2.6, 속도 ≈95px/s, 시작 스태거 0.5s,
  양끝 페이드(0→0.85→0).
- **탭 버스트**: 5~8개 입자가 34~64px 방사, 0.4s 소멸 + 오브 스프링 바운스.
- **부유(bob)**: 진폭 5~10px·주기 3.8~5.3s **오브별 랜덤**, 지연 i×0.42s.
- **트윙클 별 8개 + 먼지 입자 ~22개**가 메뉴 영역에 흩어져 명멸.
- **배지**: 어두운 유리 + 시안 림 + 글로우, 열림 시 pulse.

## 참고 사진 (파일 미보유)

원신(Genshin Impact) 푸리나가 물방울 구체에 앉아 있고, 주위로 디저트가 담긴 **발광 유리
오브들이 링(아치) 형태로 떠 있는** 스킬/컷신 장면. 오브의 유리 질감·시안 발광·주변의
반짝임 입자가 위 HTML 의 무드 원천이다.
⚠️ 사진은 채팅 붙여넣기로만 전달되어 파일이 없다 — 파일로 받으면 이 폴더에
`furina_orb_menu.png` 로 추가할 것.

## 이식 대상

`app/src/main/kotlin/com/langsense/app/overlay/` 의 `RadialOrbView`(칩 렌더),
`QuickMenuOverlayView`(선·빛 점·별·먼지·버스트·부유), `RadialMenuStyle`(모든 상수).
"저사양 모드(움직임 줄이기)" ON 이면 연속 애니메이션(모프/부유/빛 점/별/먼지)을 끈다.
